# Pokemon Card Games 對戰記錄工具

這是一個專為寶可夢卡牌玩家設計的 **純前端對戰記錄工具**，無需後端伺服器，所有資料皆儲存在您的瀏覽器中。

👉 **立即使用**：目前已將此網頁存放置在Ka Ian的Github Page靜態網頁伺服器執行。

---

## 功能特色

- **完整對戰記錄**：記錄日期、賽事、對手牌組、勝負結果、先後攻、標籤與備註
- **牌組管理**：新增、修改、刪除「我的牌組」與「對手牌組」，修改名稱時會同步更新所有相關記錄
- **賽事管理**：自訂賽事類型（預設：GYM/店家賽、高級球、大師球、紀念球、個人玩、開放賽、Live）
- **數據分析**：
  - **嬴輸和記錄**：快速查看各牌組的戰績
  - **先後攻對比**：分析牌組在先攻/後攻時的勝率差異
  - **圓餅圖**：直觀呈現各牌組的使用比例
- **分享圖片**：一鍵生成對戰記錄與勝率分析的總結圖片，方便分享
- **資料備份與還原**：可匯出 JSON 檔案備份，或匯入先前備份的資料

## 資料儲存說明

**所有資料皆儲存在您瀏覽器的 `localStorage` 中**，這表示：

| 情況 | 資料是否保留 |
|------|-------------|
| 正常關閉並重新開啟瀏覽器 | ✅ 保留 |
| 重新整理頁面 (F5) | ✅ 保留 |
| 在同一裝置使用相同瀏覽器 | ✅ 保留 |
| 清除瀏覽器歷史記錄／Cookie／網站資料 | ❌ 資料將永久遺失 |
| 使用無痕／私密瀏覽模式 | ❌ 關閉視窗後資料即消失 |
| 更換裝置或瀏覽器 | ❌ 無法跨裝置同步 |
| 手動清除 localStorage | ❌ 資料將永久遺失 |

⚠️ **重要提醒**：由於資料僅儲存於本地瀏覽器，建議您**定期使用「匯出資料庫」功能備份資料**，避免因清除瀏覽器資料或裝置遺失而導致記錄不見。

## 備份與還原操作

### 匯出資料庫（備份）
1. 捲動至頁面最底部
2. 點擊 **「匯出資料庫」** 按鈕
3. 瀏覽器會自動下載一個 `pokemon_battle_YYYY-MM-DD.json` 檔案
4. 請將此檔案妥善保存（如雲端硬碟、隨身碟等）

### 匯入資料庫（還原）
1. 捲動至頁面最底部
2. 點擊 **「匯入資料庫」** 按鈕
3. 選擇先前備份的 `.json` 檔案
4. 系統會自動載入並還原所有資料

### 刪除所有記錄
1. 捲動至頁面最底部
2. 點擊 **「刪除所有記錄」** 按鈕
3. 系統會彈出兩次確認對話框，確保您不會誤刪

## 使用說明

### 新增對戰記錄
1. 選擇日期（預設為今天）
2. 選擇賽事類型
3. 選擇「我的牌組」與「對手牌組」
4. 點擊「嬴／輸／和」選擇結果
5. 點擊「先攻／後攻」選擇先後攻順序
6. 可選填標籤（手氣好順、手氣好差、好有挑戰性、克屬）與備註
7. 點擊 **「儲存」** 按鈕
8. 儲存後，**對手牌組、結果、先後攻、標籤、備註**會自動清空，方便連續輸入

### 管理牌組／賽事
- 點擊各欄位旁的 **齒輪圖示 (⚙️)** 即可進入管理介面
- 可新增、修改名稱、刪除項目
- **修改牌組或賽事名稱時，所有相關的對戰記錄會自動同步更新**

### 查看全部記錄
- 點擊「近期對戰」區塊的 **「更多」** 按鈕
- 可依年份、月份、標籤、牌組進行篩選

### 分享圖片
- 點擊 **「我的記錄分享」** 按鈕
- 系統會生成包含近期對戰與牌組勝率分析的圖片
- 可預覽並下載 PNG 圖片

## 常見問題

**Q：更換電腦後資料還在嗎？**  
A：不在。資料僅儲存在原裝置的瀏覽器中，請務必定期匯出備份。

**Q：手機和電腦的資料可以同步嗎？**  
A：不行。各裝置的資料彼此獨立，無法自動同步。您可透過「匯出」再「匯入」的方式手動轉移。

**Q：清除瀏覽器快取會影響資料嗎？**  
A：**清除快取 (Cache) 不會影響**，但**清除網站資料 (Site Data) 或 Cookie 會導致資料遺失**。

**Q：最多可以記錄多少筆對戰？**  
A：取決於瀏覽器的 localStorage 容量限制（通常約 5-10 MB），足夠記錄數千筆對戰。
